```css
pos     position: relative;
t       top
d       display
```

### Margin & Padding

```css
m       margin
m:a     margin: auto;
p       padding
```

### Box Sizing

```css
w       width
h       height
maw     max-width
miw     min-width
```

### Font

```css
f       font

fz      font-size
ff      font-family
```

### Text

```css
ta      text-align: left;

```

```css

```

